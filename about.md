# Markdown Cheatsheet (Jekyll + GitHub Pages)

## Headings
# H1 – Page title (usually from front matter)
## H2 – Section
### H3 – Sub-section
#### H4
##### H5
###### H6

## Text Formatting
**Bold**  
*Italic*  
***Bold + Italic***  
~~Strikethrough~~  
`Inline code`

## Paragraphs & Line Breaks
This is a paragraph.

This is another paragraph.

This is a line break  
(two spaces at the end)

## Lists
### Unordered
- Item one
- Item two
  - Sub item

### Ordered
1. First
2. Second
3. Third

## Links
[External link](https://example.com)  
[Internal link]({{ "/contact/" | relative_url }})

## Images
![Alt text](/assets/img/example.png)  
![Alt text](/assets/img/example.png "Optional title")

## Blockquotes
> This is a quote  
>> Nested quote
